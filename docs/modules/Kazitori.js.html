Kazitori.js Module

Defined in: kazitori.js:1633

Kazitori.js は pushState をいい感じにさばいてくれるルーターライブラリです。
シンプルかつ見通しよく pushState コンテンツのルーティングを定義することができます。

使い方はとても簡単。

  • Kazitori を継承したクラスを作る
  • routes に扱いたい URL と、それに対応したメソッドを指定。
  • インスタンス化

example

 class Router extends Kazitori
   routes:
     "/": "index"
     "/<int:id>": "show"

   index:()->
     console.log "index!"
   show:(id)->
     console.log id

 $(()->
   app = new Router()
 )

Kazitori では pushState で非同期コンテンツを作っていく上で必要となるであろう機能を他にも沢山用意しています。
